# NexusAI — Advanced AI Chatbot

A full-stack, production-ready AI chatbot with a premium UI, built with FastAPI (Python) and Vanilla JS.

## Features
- 🚀 Real-time streaming responses
- 🌙 Dark / Light mode
- 📁 File & Image upload support
- 🌐 Live web search toggle
- 💬 Persistent chat history
- 🎙️ Microphone input
- 🧠 Multi-model support (GPT, Cerebras)

## Tech Stack
- **Frontend:** HTML, CSS, Vanilla JavaScript
- **Backend:** Python, FastAPI
- **Database:** SQLite
- **Deployment:** Docker on Hugging Face Spaces
